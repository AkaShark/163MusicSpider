# 163MusicSpider
一个获取网易云音乐歌手、专辑、歌曲、评论、歌词等数据的Python爬虫

会将获取的数据存储至mysql

---
本爬虫未使用Scrapy框架，适合初学者查看使用

###### 作者也是一名python初学者


### 文件信息
1. [artists.py](src/artists.py) 爬取所有的歌手信息
2. [album_by_artist.py](src/album_by_artist.py)  爬取专辑信息
